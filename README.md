# AI_Concept_Explainer_Chatbot
AI chatbot for explaining concepts with examples and code snippets.
Overview
AI Concept Explainer Chatbot is an educational chatbot built using Python, Streamlit, Ollama, and Llama 3.2.
It helps student understand concepts by providing simple explanations, real-life examples, and basic code snippets.

Features
- Explains concepts in simple language
- Provides real-life examples
- Generates basic code snippets
- Easy-to-use web interface
- Runs locally using Ollama and Llama 3.2

Technologies Used

- Python
- Streamlit
- Ollama
- Llama 3.2
- Requests Librarys
